First Ride
Rocky Mountain Element

Words: Henry Quinney
Photos: Margus Riga

Are we there yet? Mountain bike geometry has gone from huge strides to measured paces, and now we're at the stage where new bikes seem to be about nips-and-tucks rather than page-one rewrites.

The new Rocky Mountain Element typifies this. There have been some changes to the suspension layout and some small refinements to the geometry, but largely, it's a further evolved bike. And that's no bad thing. If you're going to gently update anything, it may as well be something as highly regarded as the incumbent Element. When released in 2021, it was met with near-universal praise as it sat front and center of the downcountry trend.

Rocky Mountain Element Details

• Wheel size: 29"
• Travel: 120mm, 130mm fork
• Carbon frame
• 65.5° head angle (neutral)
• Chainstay length: 435mm (S-XL)
• Sizes: XS (27.5"), S, M, L, XL (29")

Frame Details

The Element doesn't forget its roots, and still has water bottle mounts aplenty. On most frames, there are two on the downtube, then another on the underside of the top tube to store tools. Rocky has done a solid job of trying to maximise each size's ability to carry bottles while also having to operate within the obvious constraints. The XS frame has to make do with one 650ml bottle, the small can handle a 650ml plus a Fidlock 450ml bottle and the other sizes can handle two 770ml bottles. It's not perfect, but it's good to see them try and tailor the best solution for each size.

The bike has fully enclosed cables, which can be run either way meaning that everyone will be able to get tidy cables. Interestingly, there isn't frame storage on the Element. It has similarly slender tubes as the outgoing Element, and I can only speculate it wasn't worth ballooning the downtube, while also making it presumably heavier or weaker, or perhaps both.


Suspension

From afar this looks to have the same suspension layout also. However, if you look closer you'll see that there is no longer a bearing near the rear axle. This saves around 350 grams, which is not to be sniffed at. The rear triangle is now a swingarm that relies on a flex pivot. Rocky calls this their Smoothlink SL suspension and says that they aim for it to behave like the four-bar equivalent.

All shocks are made with size-specific tunes. That said, they all use the same rear-end size. So, while the tune might be specific, Rocky hasn't gone all in like some brands have to tune the frame size to both the kinematic and the shock.


Geometry

It wouldn't be a Rocky without a geometry adjusting flip chip, in this case the Ride-4 configuration. With it, riders can choose between 4 different settings that alter geometry and shock feel.

Ride Impressions

Riding the Element felt like something a journey back in time, if only because I rode the exact same singletrack loop that I rode 3 years ago for the last Element's first ride. How does it compare?

This bike is, broadly speaking, very similar. There is also three years difference between those rides, and memory can only do so much. The big difference that I felt was in the shock. The previous Element felt like you had to run a very high spring rate to control the shock through bigger compressions. This newer model feels a shade more progressive with more compression damping available from the shock. I can run it a lot softer, meaning it does an admirable job of hunting for grip while, thanks to the compression damping, it still retains that rocketship characteristic that made me love the other Element so much. I mainly rode it in the open compression setting, with the butterfly dial set to the firmest of the three possible positions. The Element climbs very well, and you can really plumb along smoother trails. The geometry is balanced, and gives you confidence without lethargy.

My only small gripe would the Fox Grip X damper. I'm yet to get on one and settle on a setup, and it feels inferior to the outgoing, and still excellent, Grip2 damper. It was the Grip2 equipped 34 that I rode three years ago that got my nod for suspension component of the year, something it duly won. I would take that fork over the Grip X 34.
